# Salesforce MuleSoft Management System

## 📌 Overview

This project is a backend-based management system that simulates a Salesforce-like CRM integrated with MuleSoft-style API connectivity. It demonstrates how data flows between different components using REST APIs and a structured service layer.

The system focuses on customer data management, API integration, and modular backend design.

---

## 🚀 Features

* Customer Management (Add, View, Update)
* REST API Development
* API Integration between components
* Simulation of System Integration and Data Synchronization
* Structured Service Layer for business logic
* Modular Backend Architecture

---

## ▶️ How to Run

### Prerequisites

* Java (JDK 8 or above)
* Maven

### Steps

1. Clone the repository

```
git clone https://github.com/your-username/salesforce-mulesoft-management-system.git
```

2. Navigate to project folder

```
cd salesforce-mulesoft-management-system
```

3. Run the project

```
mvn clean install
mvn spring-boot:run
```

---

## 🌐 Output

Once the application is running, access it at:

```
http://localhost:8080
```
